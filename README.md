# MguardSends
Secure, Distributed, Real-time messaging platform.

-------------------------------------------
Welcome To MguardSends.
MguardSends is a windows desktop application that enables secure communication between its users.
MguardSends was developed with privacy and security in mind. Unlike any other social apps that are being
used today, MguardSends has a different approach:
1: There is no centralized server.
  This means that nobody has full access to all of the messages. Files, messages and other useful information
  are widely distributed (breaking one does NOT mean breaking all).
2: All messages get deleted. It’s only a matter of time.
  This ensures that if any leak happens, not all messages get revealed. If you want to keep your conversations,
  copy them somewhere safe before they get deleted forever.
3: Various encryption levels.
  You can choose how secure you want to communicate. Note that even one layer encryption is a standard
  level for communication.
4: Simplicity.
  The bells and whistles of usual social apps are removed. There is only what you really need for
  communicating. No registration is needed.

-------------------------------------------
How to install: 
(Before double clicking on any of the .exe files, make sure you have the microsoft visual c redistributable.)
You start the application by running "MguardSends.exe" with 
administrator privillages. You do not have to run mSsSVC.exe
or mCsSVC.exe, the application will automatically decide if 
they need to be executed. Always launch the application in
administrator mode, or it may malfunction. You may need to allow 
the app through your firewall as well.

Available for:
windows operating system - x86 , x64
series : xp , 7 , 8 , 8.1 , 10 , 11

Programming Environment: 
MSVC + Win32 API
